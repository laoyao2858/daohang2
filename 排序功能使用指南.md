# 导航站新增功能使用指南

## 🎵 自定义音乐播放器

### 后台管理
1. 登录管理员面板
2. 找到"自定义音乐管理"区域
3. 点击"添加音乐"按钮
4. 填写音乐信息：
   - 音乐标题（必填）
   - 艺术家（可选）
   - 音乐文件URL（必填，支持mp3等格式）
   - 封面图片URL（可选）
5. 保存后音乐将添加到播放列表

### 前台功能
1. **自动播放**：根据设置自动播放音乐
2. **手动控制**：
   - 播放/暂停按钮
   - 上一首/下一首按钮
   - 进度条显示
3. **迷你播放器**：左下角的迷你播放器可以收起/展开

## 🎨 网站个性化设置

### 可设置项目
1. **网站标题**：显示在浏览器标签和页面头部
2. **网站描述**：显示在网站头像下方
3. **网站头像**：圆形头像图片
4. **背景图片**：全屏背景图片
5. **欢迎语**：分为标题和正文两部分
6. **自动播放音乐**：开关控制
7. **显示访问统计**：开关控制

### 设置方法
1. 登录管理员面板
2. 找到"站点个性化设置"区域
3. 修改各项设置
4. 点击"保存设置"按钮

## 📊 访客统计系统

### 实时数据
1. **总访问量**：网站总访问次数
2. **今日访问**：当天访问次数
3. **在线用户**：5分钟内活跃的用户数
4. **独立访客**：基于session的独立访客数
5. **访客分布**：按地理位置统计

### 查看方法
1. 点击右下角的统计图表按钮
2. 或通过侧边栏的"Visitor Stats"菜单
3. 统计面板会显示实时数据

### 管理员查看
1. 登录管理员面板
2. 找到"访问统计数据"区域
3. 查看详细统计数据和最近访问记录

## 🔧 原有功能升级

### 分类管理
- 拖拽排序功能更加流畅
- 支持分类类型切换（侧边栏/顶部栏）

### 网站管理
- 添加了标签功能
- 支持网站分组

### 搜索功能
- 支持中文拼音搜索
- 支持多搜索引擎切换

## 🔄 数据备份与恢复

### 导出数据
1. 登录管理员面板
2. 点击"导出数据"按钮
3. 系统会下载一个JSON文件

### 导入数据
1. 登录管理员面板
2. 点击"导入数据"按钮
3. 选择之前导出的JSON文件
4. 系统会自动恢复所有数据

## ⚙️ 注意事项

1. **音乐文件**：建议使用稳定的CDN链接，避免404错误
2. **图片文件**：建议使用WebP或JPEG格式，优化加载速度
3. **访问统计**：数据每30秒自动更新一次
4. **浏览器兼容**：建议使用Chrome、Firefox、Edge等现代浏览器
5. **移动端优化**：部分特效在移动端会适当简化以提升性能
