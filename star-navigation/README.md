# 🌟 星海导航 - 星际主题导航网站

一个现代化、功能丰富的导航网站，构建于 Cloudflare 平台。

## ✨ 特性

- 🚀 **星际主题设计** - 动态粒子背景 + 科幻UI
- 🤖 **AI智能助手** - 集成AI对话功能
- 🌤️ **实时天气** - 显示当前位置天气
- 📊 **数据统计** - 网站访问量分析
- ⚡ **极速加载** - 基于 Cloudflare 边缘计算
- 📱 **响应式设计** - 支持所有设备
- 🔐 **安全可靠** - HTTPS + 数据加密

## 🏗️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **后端**: Cloudflare Workers + Pages Functions
- **数据库**: Cloudflare D1 (SQLite)
- **存储**: Cloudflare KV
- **部署**: Cloudflare Pages
